Font InstallerのRuby版です。

## 実行方法
管理者権限で実行してください。

sudoなどでもいいです。

```sh
bundle install
ruby ./install.rb [フォント名]
```

## 依存関係

- Ruby<br>[Rubyのホームページはこちら](https://www.ruby-lang.org/)

- bundler(Rubyライブラリ)
```sh
gem install 'bundler'
```
