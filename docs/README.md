The Ruby version of the Font Installer.

## How to run
Run it with administrative privileges.

You can also use sudo.

```sh
bundle install
ruby ./install.rb [font name]
````

## Dependency

- Ruby<br>[Ruby's website is here](https://www.ruby-lang.org/)

- bundler(Ruby Library)

```sh
gem install 'bundler'
```
